# Task Manager API

A simple REST API to create, update, delete and list tasks.

## Tech Stack
- Python
- FastAPI
- SQLite
- Git

## Features
- User authentication
- CRUD tasks
- JSON responses

## How to run
1. Clone the repo
2. Install requirements
   `pip install -r requirements.txt`
3. Run
   `uvicorn main:app --reload`
